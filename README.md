## About The Project

[![Product Name Screen Shot][product-screenshot]](https://github.com/KaranamLokesh/Neighborhood-characteristics-navigation)

This is a project aimed at creating navigation based on neighborhood characteristics using Google streetview.
Here's why:
* Google streetview provides us access to a lot of images and we can analyze them to get information regarding neighborhoods.
* We can learn about the neighborhood by studying some features from it.
* Our goal is to get images from the neighborhood and analyze the characteristics using Deep Learning.
